What is a Spreadsheet?

Spreadsheets could be called number processing software as opposed to Word, which is word processing software. All spreadsheet software (including Excel, Lotus and QuattroPro) is based on a table format – numbers (and text) are entered into cells, which are organized into rows and columns. A spreadsheet can be used to:

  • Format data or calculations into an attractive table;
  • Perform complicated or repetitive calculations;
  • Analyze empirical (laboratory) data and determine the equation that best fits the data;
  • Graph data in many types of graphs;and
  • Store data in a database (although Excel is not as versatile or powerful as the Microsoft Access database program).

Examples:

A spreadsheet would be suitable for the following applications:

  • Calculating your tax return;
  • Preparing a cost estimate for a construction project;
  • Recording and calculating your marks on tests and assignments and then calculating each students final grade;
  • Estimating the flow rates in a drainage or sewerage analysis and calculating what pipe diameter would be required to pass the flow;
  • Calculating mortgage payments given the principal, interest rate and number of payments, calculating the balance owing after each payment;
  • To count the number of days / minutes / seconds until the year 2010 (or any other date)
  • Calculating the amount in an investment account each month for 20 years;
  • Calculating and graphing stress and strain and graphing the results in a lab; and
  • Performing statistical analysis (such as calculating the standard deviation) of a large number of data points.

Spreadsheet Glossary
Worksheet
A single table of cells.
Workbook
A collection of 1 or more worksheets. Usually stored in a single file.
Chart
Also known as a graph. A representation of data. Can appear either as an Embedded Chart on an existing table, or in its own Sheet.
Cell
A field on a table that can contain data. Represented by the intersection of a row and column. Eg: C3
Range
A collection of cells. Represented by the address of two cells in diagonally opposite corners. Eg: B5:D1
Formula
A mathematical operation performed in a cell. Often reference other cells for input data. Eg: =(A1+A2)*B3
Reference
A way to fetch the data from one or more cells. Used by supplying the cell or range address in a formula. Eg: C3, A1:D4
Function
A predefined and reusable set of operations. Translates input into output. Often performed on a referenced cell or range. Can be part of a formula or embedded in other functions. Eg: =SUM(C3:C6)
Auto Fill
An action performed in Excel to intelligently fill cells with new data based on the data in other selected cells. Accessed using the handle in the bottom-right corner of a cell.
Spreadsheet Structure

A Worksheet is a single table within a Workbook. Using several Worksheets makes it easier to organize different types of data. A Workbook is saved as a single .xlsx (.xls in older versions) file and all Worksheets within the Workbook are opened when you open the file.

Worksheets are separated by Tabs visible at the bottom of the screen. Tabs are initially called Sheet1, Sheet2, Sheet3 but you can rename them by double clicking on the Tab. You can add more sheets by clicking the plus to the right of the existing tabs, or remove them by right-clicking a tab and selecting Delete.

Columns are labeled with letters; Rows are labeled with numbers; Cells are identified by their column and row, for example: A1, B265, G4, AA123 all the way to XFD1048576. These notations are called the cell’s address. This address can be extended further by including the name of the worksheet the cell resides in. For example: worksheet_name!A1.

Cell References

It is very common in Excel to perform an operation using the data of other cells. To do this, you must provide a reference to the cell or cell range. This is done simply by typing the address of the cell(s) directly into the formula where the data is needed.
