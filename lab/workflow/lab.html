    File Management Basics

    You can organize the files on your computer and on connected network drives using the File Explorer program in Windows. You can open this application by double-clicking on any folder on your desktop (or the My Computer icon), or opening it from the Start menu or task bar.

    The left-hand sidebar of every Explorer window shows a collapsable list of common locations. The Computer section includes all attached drives and devices, and the Network section includes drives available over the network. Your "M" drive (often written "M:\") is a network drive.

    The main content area of an Explorer window shows the contents of a single selected folder. If you've selected your M drive from the sidebar, the main area will show the files stored in your M drive. In general, you can drag and drop files into this area to store them in the selected folder, or drag files out to remove them. You can also use standard cut & paste commands to move files in the same way. You can create copies by holding down the Ctrl key while dragging, or by using copy instead of cut commands. Because your M drive is a network drive, dragging files in and out of it will copy files instead of moving them by default; use Shift to reverse this.

    To create a new folder, click the New Folder button near the top of the Explorer window. Immediately after pressing the button, the folder name will be highlighted and can be changed simply by typing the name you want and pressing enter. If you'd like to change its name later, right-click on the folder and choose Rename from the contextual menu.

    Excel Functions

    If a cell's content starts with an equals sign "=", it will be interpretted as a function. Functions compute a value to be displayed using math and logic. For example, "=2+2" would compute the result of 2+2 and display 4 as the content of the cell.

    The BASE function transforms a decimal number into a number in another base system (such as binary or hexadecimal). To use it, type =BASE(, followed by the number to convert, a comma, the number of the base system to convert to (eg. "2" for binary, or "16" for hexadecimal), and a closing ). For example, "=BASE(10, 16)" would result in "A" (10 in hexadecimal). Instead of entering the number to convert explicitly, you can specify a cell that contains the number you want: for example, "=BASE(A1,2)" would convert the number found in cell A1 into base-2.

    Cells containing functions can be "filled-down" in the same way as regular values, and cell references will be automatically adjusted. For example, if column A contains numbers to convert and you enter =BASE(A1,16) in the first cell in column B and then fill-down, the A1 reference will be adjusted to target the cell in the same row as each filled cell (eg A2, A3, etc.).

    What is CSV

    CSV ("Comma-separated values") is a plain-text format for storing tabular data. Each row of the table is stored on a single line in the file, and each column within a row is delimited using commas. Because of this file type's simple structure, individual cells may not contain multiple lines of text and may not contain commas.

    A closely-related format, TSV, uses tabs instead of commas to delimit columns. This allows cells to contain commas, intead prohibiting them from containing tabs.

    More about file names and extensions

    Two files in one folder may never have the same name. Windows will warn you that it will overwrite an existing file if you attempt to save a file with a name that is already in use. However, the names of files usually also include a "file extension" that specifies the file's type, and so two files may have the same name if their types (and extensions) are different!

    Windows normally hides file extensions by default, since they are important for distinguishing the file's type and should not normally be changed by hand. However, it is sometimes useful to be able to see these type extensions in order to differentiate files with similar names.

    To show file extensions in Windows 7,

    1. Click the Organize button in any Explorer window
    2. Choose Folder and Search Options from the pop-up menu
    3. Switch to the View tab in the settings window that opens
    4. Scroll down to the Hide extensions for known file types option and un-check it.

    To show file extensions in Windows 8 and up,

    1. Click on the View ribbon
    2. Check the File name extensions box.

    As mentioned in the Paste Options above, linked objects will keep a reference to the original file they're based on, and update if there are changes.

    Bitmap encoding

    The logo.bmp image is encoded in an uncompressed format which is simple enough to be partially read and manipulated by hand. The file starts with some metadata that describes the format of the image (and which is not easily understandable), but the majority of the file is composed of sequences of color values ranging from 00 to FF as you've seen in lab assignment 5.

    Images in general are made up of a series of dots called "pixels" (short for "pixel elements"). In a "24-bit bitmap" image like this one, each pixel is recorded one after the other in the file, with each pixel's colour described by three 8-bit numbers (2 hex digits each) for the amount of red, green, and blue.

    Most image formats you may be familiar with (such as GIF or JPEG) do not encode their colour values quite so simply, and so would not be at all understandable in a hex-viewer. These image formats make use of compression to reduce the size of the file by removing redundant information.
