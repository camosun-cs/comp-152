Instructions for uploading files to the server.
 
For the HTML, Javascript, and PHP labs you will need a file transfer program that can use either sFTP or SCP. For example, Filezilla and WinSCP. You can download one of these programs at:
http://sourceforge.net/projects/winscp/
http://filezilla-project.org/
 
In the labs you can use the SSH program which does SCP. You can find it under START >> All Programs >> Computing >> Filezilla FTP Client.

Using the SCP/sFTP client log on using:
hostname: studentlabs.cs.camosun.bc.ca
username: same as your D2L login
port: 22
password: send me an email and I will let you know what it is (this will give me a pretty good idea of when you started on this lab)
File Protocol: SCP or sFTP
 
After you log on you will see a folder in the right panel called public_html. Double click on public_html to enter that directory. This is where all your web files should be transferred to when you are done creating the pages. Create a new directory called images. This is where all the images that you use for your webpages hould go. When you refer to them in your html code, you should include the path "images/" before the filename.
Do not forget to set the permissions/properties of all your files. It should be:
Owner: read, write, execute
Group: read, execute
Other: read, execute
 
Additional IMPORTANT information

To view your web page in a browser, type into the address bar the following:

http://studentlabs.cs.camosun.bc.ca/~user.name##

  1. user.name## is the same as you D2L username.
  2. NOTE the forward slashes (/).
  3. Linux systems are care sensitive. (So is your username and password)
  4. The browser will display the index.html file in a directory if it is there, by default.
  5. While working on your web pages, save your work often on your machine. Transfer the files to the web server to check your work.
  6. IF YOU FIND ERRORS on your website and want to fix them, fix them on your computer and copy them back to the web server.

Please test this before you begin your lab. If you have any questions, please post them in the water cooler.
