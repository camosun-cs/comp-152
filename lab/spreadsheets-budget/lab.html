    Creating formulas

    Along with text, cells can execute and display mathematical formulas. All formulas in Excel must start with an = sign, and, like text, can be entered directly into the cell or into the formula bar below the ribbon.

    Excel operators

    =
    equals.
    + (shift + =)
    addition.
    -
    subtraction.
    * (shift + 8)
    multiplication.
    /

    Formulas follow the BEDMAS order of operations

    A cell address can be used in any part of a formula to use that cell's displayed value instead, even if that cell is in itself another formula. To insert an address, select the desired cell or cell range with the mouse while the formula is still being edited. The address will be added at the text pointer's position. The address can also be typed in manually.

    A cell address consists of a Column letter(s) and a Row number. For example: A1, F28, GB102.

    For a group of cells, use the address of the two cells in opposing corners of the box selection, with a : in between. Example: A1:F2, X50:B2

    The act of fetching the values of other cells is known as a reference. References will automatically update their values if the cell they're referring to changes, which will then update the result of your formula. You can see this in action by changing one of the individual expense fields.

All About Autofill

Now that the required formulas and styles are working for one month, it's time to fill out the remaining months. This budget table will only cover September to December, but these principles will work on any table of any length.

  1. Begin by using autofill to fill out the remaining 3 months. Keep an eye on the helpful tooltip that displays the last result of the fill (Dec here).

    Autofill basics

    Selected cells will have a small box in the bottom-right corner of the selection area. If you click and drag this into adjacent cells, Excel will intelligently generate data based on the selected content into the new cells.

    For example, if the selected cells contain the first few numbers of a simple pattern (such as three cells with 0, 1, and 2), Excel will generate more numbers that follow the sequence.

    If the selected cell contains a formula, that formula will be filled into the new cells, and any relative references will be automatically updated to point to the next cell in the direction being filled in.

    Any formatting or styles applied to the source cell will also fill over by default.

  2. The next three rows will likely be changing each month, so they don't need their results copied over, only the way they're formatted. Fill them to the right as well, and set them to Fill Formatting Only.

    You can autofill these three rows at the same time. Just box-select the desired source cells first. Remember: the source cell must be the data, not its header.
    Autofill Options

    After every fill operation, a small button will pop up in the bottom right corner of the selection. Clicking this reveals options to apply to the previous fill.

  3. The remaining four rows (Rent, Total Expenses, Income, Net Income) don't need to follow a pattern like the month headers, nor remain empty like the expenses. Rent and Income are (hopefully) consistent each month, and the formulas need to apply to each column. Use the Copy Cells autofill on these remaining rows to satisfy these requirements.

    The default action, Fill Series, would still work as expected with formulas and functions. It copies them over to the new cells and updates the references accordingly. Using Copy Cells just lets us to fill all these rows at once. Don't worry about including the blank row in the initial selection; a copy of nothing is still nothing.

  4. All that remains is to add the rest of your expenses to the budget table. If everything has been done correctly, the Balance field should slowly deplete as you input more expenses. Did you stay within your budget?

    As mentioned earlier, the Enter key advances to the cell below the current cell. Similarly, the Tab key advances to the next cell to the right. A trick involving both of these shortcuts is that, after using Tab to fill in multiple cells in a line, pressing Enter will advance to the beginning of the next line, right below the first cell you started on. See if you can quickly fill in all the empty expense fields, starting with C2, without touching the mouse.

Worksheets and Workbooks

A Worksheet is a single table within a Workbook. Using several Worksheets makes it easier to organize different types of data. A Workbook is saved as a single .xlsx file and all Worksheets within the Workbook are opened when you open the file.

Worksheets are separated by tabs visible at the bottom of the screen. A new Workbook may already have three sheets called Sheet1, Sheet2, and Sheet3, but you can rename them by double clicking on the tab. You can add more sheets by clicking the plus to the right of the existing tabs, or remove them by right-clicking a tab and selecting Delete.
