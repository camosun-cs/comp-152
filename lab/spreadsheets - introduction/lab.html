What is a Spreadsheet?

Spreadsheets could be called number processing software as opposed to Word, which is word processing software. All spreadsheet software (including Excel, Lotus and QuattroPro) is based on a table format – numbers (and text) are entered into cells, which are organized into rows and columns. A spreadsheet can be used to:

  • Format data or calculations into an attractive table;
  • Perform complicated or repetitive calculations;
  • Analyze empirical (laboratory) data and determine the equation that best fits the data;
  • Graph data in many types of graphs;and
  • Store data in a database (although Excel is not as versatile or powerful as the Microsoft Access database program).

Examples:

A spreadsheet would be suitable for the following applications:

  • Calculating your tax return;
  • Preparing a cost estimate for a construction project;
  • Recording and calculating your marks on tests and assignments and then calculating each students final grade;
  • Estimating the flow rates in a drainage or sewerage analysis and calculating what pipe diameter would be required to pass the flow;
  • Calculating mortgage payments given the principal, interest rate and number of payments, calculating the balance owing after each payment;
  • To count the number of days / minutes / seconds until the year 2010 (or any other date)
  • Calculating the amount in an investment account each month for 20 years;
  • Calculating and graphing stress and strain and graphing the results in a lab; and
  • Performing statistical analysis (such as calculating the standard deviation) of a large number of data points.

Task 1: Powers of 2, Powers of 10

Introduction

Computer memory is measured in bytes, which is the amount of memory historically used to store a single character. Because computers use a binary number system (with only two digits: 1 and 0), larger amounts of memory are often measured internally by power-of-2 multiples of bytes. For example, the NTFS filesystem usually stores data in chunks of 4 "kilobytes" (4 × 210 bytes), and modern RAM modules are usually measured in multiples of 230 ("giga") bytes.

On the other hand, people usually measure things in the real world using SI prefixes, which are based on powers of 10. For example, we measure distance using kilometers (103 meters) and power using gigawatts (109 watts).

Confusingly, the prefix system for powers of 2 used in computing uses the same names as those in SI, even though they refer to slightly different numbers! As you can see in the figure below, the base-2 and base-10 prefixes are fairly close approximations of one another, but they diverge more and more as the prefixes get larger.

Conventions exist that help to distinguish base-2 from base-10 prefixes (eg. KiB vs. KB, or kB vs. KB), but they are rarely used in practice.

If that weren't confusing enough, CPU speeds and hard-drive sizes are usually measured and marketed using SI prefixes, while other computer components continue to be marketed using base-2 prefixes!

In this task, you will be using Excel to compute powers of 2 and 10 up to the 40th prefix value and create a graph of their divergence.

    Using Autofill

    Autofill is an action performed in Excel to intelligently fill cells with new data based on the data in other selected cells. For example, if the selected cells contain the first few numbers of a simple pattern (such as three cells with 0, 1, and 2), Excel will generate more numbers that follow the sequence.

    To use autofill, select the cell or cells from which to you want to generate generate the patter. A small box will appear in the bottom-right corner of the selection. Click and drag from this box into one or more adjacent cells to fill into them.

    If the selected cell contains a formula, that formula will be filled into the new cells, and any relative references will be automatically updated (more on references below).

    References

    To use the value of a cell inside another cells formula, you must use a reference. To create one, type the cell's index into the formula (ex: A1), or just click on the cell to automatically insert it.

    By default, this will be a relative reference. This means that when autofilling this formula down into other cells using the fill handle, the references will automatically change themselves relative to each cell being filled. For example, if a formula in cell A1 is referencing cell C1, filling that formula down the column will update each cell's reference to target the cell in its own row, A2 will reference C2, and A3 will reference C3.

    Absolute References

    As mentioned earlier, using a plain cell address in a formula creates a relative reference, which is dynamically updated when using autofill. If, instead, the reference needs to always target the same cell, as is the case when referencing a single constant number, make it an absolute reference by prefixing the row and/or column character in the address with a $, for example, $B$5. If either the row or column address is left un-prefixed, the reference will still be relative in that direction only.
