Task 1: Sections and Page Layout

  1. Download this unformatted document, rename it file WordProcessing_First_Last.docx (using your first and last name), and open it in MS Word.

  2. Create the title page by positioning the text cursor at the start of the line "1. Introduction", then inserting a section break.

    Section Breaks

    A document can be divided into Sections using Section Breaks. Sections are used to create layout or format changes. For example, one section could contain a full page centered title, while another could span the body text across multiple columns.

    Individual sections can have these formats changed:

    • paper size and paper source
    • page orientation
    • margins
    • page borders
    • headers and footers
    • page numbering
    • footnotes and endnotes
    • number of columns

    To insert a section break:

    1. Place the text cursor where you would like the break to be inserted (text after the cursor will be placed into the new section).
    2. Click the Breaks drop-down menu is in the Page Layout tab.
    3. Select the desired section break. Be careful not to pick a Page Break (the top 3 selections), as this will not create a new section. For most cases, you will want to create the section on a new page.

  3. Change the page layout of the first section (the title page) to increase the size of the vertical margins and vertically center its content.

    Page Setup

    Open the Page Setup window, found in the Page Setup section of the "Page Layout" tab. This window contains options for changing print Margins, Paper size, and Layout. The Vertical Alignment drop down is found under the Layout tab, in the Page section.

    Take note of the Apply to:, which determines where the page formats will take effect. The default, This section, applys the format to the section the text cursor is currently in.

  4. Add automatically-generated page numbers that begin on the second section, skipping the title page.

    Advanced Line Numbering

    By default, a section will use the same header/footer as the previous section. This is why adding text or objects to a page header adds it to all the headers in the document.

    To create page numbers that skip a section, this behaviour must first be overridden. There are two ways to go about this.

    • Double-click in the header/footer area of a page in the section you want to add elements to (not the section you want to skip) to open the Header & Footer Tools: Design tab in the ribbon.
    • To only omit the first page from the line numbering or other header elements, check the Different First Page option.
    • In all other scenarios, click the Link to Previous button in the Navigation section to disable it.
    • Page numbers can now be added using the dropdown in the Header & Footer section.
    • To start the numbering from 1, select Format Page Numbers... from the Page numbers dropdown and adjust as needed.

    Styles

    In MS - Word, a style is a collection of formats saved together and given a name. Apply the same style to several paragraphs and they will all match. Once a style has been defined in a document, it can be used in several places to quickly ensure consistency of formatting. A style can later be re-defined to have a different appearance, so that all text with that style will automatically take the new appearance.

    A style can include:

    • font formatting (font, size, bold/italic/etc, colour)
    • paragraph formatting (alignment, indents, space within and between, page break control etc.)
    • tab stop positions
    • borders and shading
    • bullets and numbering
    • language

    A number of useful styles are built-in with MS Word, and these are available for immediate use.

    • A style is most easily applied using the Styles Gallery – found in the Styles group of the Ribbon’s Home tab. Each style is displayed with a small preview such as
    • A few popular styles are displayed, and others can be found by scrolling the list.
    • Pausing the mouse over a style in this gallery will display the current (selected) text as it would look with this style applied.
    • If the style seems suitable, you can click the style name to apply it permanently.

    For headings, sub-headings and minor headings, use the styles called Heading1, Heading2 and Heading3 respectively. When all headings are formatted using these styles, you can be sure that equivalent headings will match – for example, all Heading2 paragraphs will look the same.

  3. Right-click the Normal style again and select Modify. Set the font format to something other than Times New Roman. Choose a clean, readable font. For example: Calibri, Georgia, or Minion Pro. Click OK, and Normal will be updated to use the new font.

    Bullets and Numbering

    Highlight each list, then click the Bullets or Numbering buttons in the Paragraph section of the "Home" tab. Word will usually do a pretty good job of detecting bullet types and sub-lists.

    The bullet style can be modified by clicking the down arrow next to the applicable list button.

    The Increase and Decrease Indent buttons move the selected list item up or down the list hierarchy, or indent the entire list left or right if the top item is selected.

  7. Create an attractive visual theme.

    The Design Tab

    The Design tab contains Themes, Style Sets, Colors and Fonts

    Notice how changing any of these options automatically changes the formatting of all the text in the document. This is because it is modifying the Styles you applied to that text.

    Using Draft View

    Because they're normally invisible, section, page, and line breaks can be tricky to deal with. You can use Draft View to help with this.

    After enabling Draft View (found in the View tab in the ribbon), section breaks will be displayed as a horizontal double line. To reliably delete a break, click on it with the mouse to position the text cursor on it, then press the Delete (or Del) key on your keyboard.

    Click Print Layout in the View tab to return to the default view.

    Table of Figures/Contents

    If your document has been formatted consistently, Word can automatically generate organizational tables. These tables can also be later updated if the document changes.

    With the Table of Contents or Bibliography, clicking inside the table will reveal an Update Table option above it.

    The option to update the Table of Figures is in its context menu (right click in it). You can also update it with the button in the References tab of the ribbon.

    Custom Styles

    To modify an existing style (thus affecting all text using that style):

    1. Apply additional formatting rules to some text with that style.
    2. Select it.
    3. Right-click the style in the Styles section of the ribbon, and click "Update Style to Match Selection".

    To create a brand new style:

    1. Add the desired formatting to some text.
    2. Select it, then right-click it.
    3. Click the Styles box that appears above the context menu.
    4. Choose "Create a Style".
