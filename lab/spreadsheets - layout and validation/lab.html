Learning Outcomes and Introduction

In this lab assignment you will learn about:

  • spreadsheet layout and formatting
  • finding data with VLOOKUP, MATCH, and INDEX functions
  • conditional formulas using the IF function
  • computing totals with the SUM function
  • manipulating text with UPPER, CONCATENATE, RIGHT, and REPT functions
  • data validation

Task 1: Create an order form (15 marks)

  1. Create a folder in the COMP152 directory called Lab 9. All further work in this assignment will be done in this folder.
  2. Download this data file
    3. Set Up Environment
    You may also want to create applicable directories to organize all your other work done so far in this course.
    At this point, you should be comfortable with organizing your work in files and managing your data.
  3. Open Microsoft Excel, and create a new blank workbook.
  4. Create a new sheet in the workbook and name it Invoice
  5. Add the heading COMP152 Sales Invoice at the top of new sheet.
  6. Just beneath the main heading, add a Date label and space to record a date.
  7. Add a Customer label with space for five rows of content.
  8. Beneath that, layout a table with five rows and six columns labelled:
    • Product
    • Description
    • Cost
    • PPU
    • Quantity
    • Tax
    • Total
  9. Add labels and data entry space for Subtotal, Taxes, Credits, Amount Due, and Payment Method. Make sure to leave at least two blank cells next to payment method.
  10. Add your style for wow factor.
    Style

    Use borders, backgrounds, font-styles, and cell-merging to differentiate labels from input fields and make the form look appealing.

    Arrange the elements of your form to clearly show the relationship between fields (eg. Subtotal will eventually show the sum of amounts in the Total column) and ensure there is enough space in each input field to for the data it will likely hold.

    A large part of your "Wow factor" mark will come from this step; don't gloss over it!
  11. Format your order form
    Formatting
    1. Apply currency formatting to every every cell that will show a dollar amount (eg. Cost, Tax, Subtotal, etc.).
    2. Apply a YYYY/MM/DD date formatter to the date input cell.
  12. Use the VLOOKUP function to automatically fill the Description and PPU fields of the invoice based on the given Product code and the list of merchandise on the Products sheet.
  13. Add a formula to calculate the value of the Cost column for each line item by multiplying the PPU by the Quantity. Use the IF function to apply a 5% discount if more than 10 units of the same type are ordered.
  14. Use the SUM function to automatically fill the Subtotal field with the sum of the Cost column and the Taxes field with the sum of the Tax column.
  15. Use a formula to fill the Amount Due field with the value of Subtotal plus the value of Taxes less the value of Credits.
  16. Use a combination of IF and VLOOKUP to show the customer's credit-card number next to the Payment Method field if the chosen method is Credit Card.
  17. Check that all of your formulas work correctly by adding two or three products to your invoice.
    What happens if you need to record more than 5 line items?

Deliverables

  • Modified spreadsheet with a working invoice on a new sheet.

Task 2: Validation and Text manipulation (10 marks)

  1. Add a data validator to the Payment Method input field to allow only Credit Card, Cash, or e-Transfer.
    Hint: You should create a new sheet to do this cleanly!
  2. Add a data validator to the first line next to the Customer input field to ensure it is the name of an existing customer from the Customers sheet.
  3. Use VLOOKUP to fill in the street address, city, province, and postal code of the customer in the spaces below their name.
  4. Use the CONCATENATE function to combine the customer's city and province into a single cell, separated by a comma (eg. "Toronto, Ontario").
  5. Use a combination of MATCH, INDEX, and UPPER to add the customer's company name in capital letters.
    Because the company name is to the left of the customer name, you won't be able to use VLOOKUP for this!
    At the end of this step, your data next to the customer field should look something like this: 
    Mr. Peabody James
INLAND TECHNOLOGY INDUSTRIES
1234 Nowhere Ln.
Toronto, Ontario
M4N 9N8
  6. Use REPT, RIGHT, and LEN to adjust the formula for the customer's credit card number so that only the last 4 digits are shown and all other digits are replaced with X's (eg. XXXX-XXXX-XXXX-0305).

Deliverables

  • Modified spreadsheet with automatically-filled customer details, reformatted credit card number, and appropriate validators.
