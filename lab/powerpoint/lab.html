Learning Outcomes and Introduction

In this lab assignment you will learn about:

  • Using the slide master to style your slide show
  • Embedding and formatting media
  • Recording and presenting your slide show

Slide Master

  1. Enter Slide Master view by clicking Slide Master in the View ribbon.

    About Slide Masters

    The slide master is the template slide from which new slides get their layout and styles. Using slide masters properly will not only save you time, but also keep your presentation consistent.

    Each slide master can contain several slide layouts. A layout defines a default set of elements and their position. When you begin creating your slide show, each layout will appear in the New Slide dropdown in the Home tab.

    If set up properly, each slide in your presentation will inherit its style and elements from a layout which will, in turn, inherit from a single slide master. This means you only have to create any style or layout once, and then reuse it for each applicable slide. You never need to manually edit an individual slide element's style.

  2. Select the slide master if it isn't already and create a consistent style for your presentation.

    Themes

    PowerPoint comes with several themes, available in the Themes dropdown. These change the look of everything, including fonts, colors, and backgrounds.

    After picking a theme, or sticking with the default plain and simple Office Theme, you can edit each aspect of the theme yourself. Just make sure you only edit in the slide master, so you won't have to repeat yourself on each new slide.

    • Fonts, colors, and effects can be changed in the Background section in the Slide Master ribbon.
    • Individual elements (text, images, etc.) can be styled by first clicking the element, then opening the Format ribbon.
  3. To change any of the default included layouts, select them under the slide master. New elements can be added using the Insert Placeholder dropdown.
  4. New layouts can be created with the Insert Layout button in the Edit Master section. They can then be renamed or deleted using the buttons in the same section, or by right-clicking them and selecting the applicable option.
  5. If you need to have multiple styles in the same presentation, multiple slide masters can be used. Use the Insert Slide Master button. The new style and its layouts will appear in the New Slide dropdown. This is usually not necessary for short presentations like this assignment.
  6. Once you're happy with the look of your presentation, exit Master View using the close button in the ribbon. Adding new slides using the dropdown in the Home ribbon will now show your styled layouts.
  7. As long as you don't override a slide's inherited style by manually adjusting or styling its elements (which you should never do if you plan to do it more than once), you can edit the slide master at any time to change its style.

Embedded Media

  1. The Insert ribbon contains buttons for embedding pictures, videos, and audio in your slide show.

  2. After an image or video has been inserted, it can be resized and positioned by first selecting it, then using the drag handles. Use the corner handles to keep the element's proportions correct

    Tip: you can hold the control button down while resizing to keep the element centered.
  3. The Format ribbon (available after selecting an element) contains options for styling the element, as well as precise positioning options, including a useful Align dropdown in the Arrange section.

Presenting Your Slide show

  1. Notes can be added to each slide by using the Notes Pane, the button for which is found in the View ribbon or in the bottom status bar. Notes added here will be shown to you while presenting, allowing you to script out your commentary.
  2. To start a slide show for live presenting, use the options in the Start Slide Show section of the Slide Show ribbon.

  3. A slide show can also be set up to play automatically along with recorded commentary using the Record Slide Show option.

    Recording Slide Shows

    The Record Slide Show option saves slide timings (how long each slide is displayed) and can also record your voice for fully automated presentations.

    Upon starting a slide show recording, you can use the Presenter View window that pops up to give your presentation the same way you would in a live setting. However, you also have the option to re-record just the current slide if you make a mistake.

    You can delete recorded audio and timings using the Clear option in the Record Slide Shop dropdown.

  4. To play the recorded presentation, ensure the Play Narrations and Use Timings options are checked, then start the slide show normally. If you are unhappy with the results, recording again will overwrite the previous result.

    If you need to do a live presentation with manual slide control, un-check Play Narrations and Use Timings.
  5. If you now save your project as a PowerPoint Show: .ppsx (as opposed to the default .pptx), the resulting file will contain everything it needs to run your presentation anywhere with PowerPoint installed. Double-clicking this file will immediately launch and start the presentation. You can still open and edit .ppsx files normally by opening them through PowerPoint.
